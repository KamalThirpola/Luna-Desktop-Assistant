# 🌙 Luna AI Desktop Assistant

Luna AI Desktop Assistant is a desktop-based AI assistant built with Electron and JavaScript. It allows users to interact with Google's Gemini AI model through a simple and intuitive desktop interface.

## Features

- AI-powered chat assistant
- Desktop application using Electron
- Fast and lightweight interface
- Google Gemini AI integration
- Modern UI for conversations

## Tech Stack

- Electron
- JavaScript
- HTML
- CSS
- Vite
- Google Gemini API

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file:
   ```
   GEMINI_API_KEY=YOUR_API_KEY
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Project Status

This project was built as a learning project to explore AI integration with desktop applications. Future improvements include voice input, chat history, and additional AI-powered desktop features.
